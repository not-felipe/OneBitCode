Formas de Armazenamento

  • Session Storage
    • Armazena dados durante a navegação
    • Persiste enquanto a aba estiver aberta, mesmo após muda r de janela
    • Existe somente enquanto a aba está aberta, após fechar os dados são limpos
  • Local Storage
    • Armazena mesmo que a aba seja fechada, sendo excluída somente quando o usuário desejar
    • Não possuem data de expiração
  • Cookies
    • Dados armazenados em pequenos arquivos de texto no computador do cliente
    • Podem possuir data de expiração
    • Forma mais tradicional de permitir "lembrar" de informações mesmo após fechar o navegador
    • São enviados nas requisições do navegador e podem existir no servidor
